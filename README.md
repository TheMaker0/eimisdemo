## 📌 Features

*   **Key Performance Indicators (KPIs):** Real-time tracking of total items, employee counts, purchase order values, and pending requests.
*   **Trend Analytics:** Visual charts for monitoring Requisitions vs. Payments and live Stock Card Movements.
*   **Inventory Distribution:** Clear breakdown of assets by categories (Office Supplies, IT Equipment, etc.) and item status (Consumable, Semi-Expendable, Property).
*   **Recent Activity Log:** Live transaction tracking showing item statuses like "Approved" or "Issued".

## 🛠️ System Modules

The application navigation is split into four core areas:
1.  **Dashboard & Employees:** Overview metrics and staff management.
2.  **Inventory Management:** Dedicated spaces for Fund Clusters, Supplier Registries, and Property lists.
3.  **Analytics:** Report generation tools.
4.  **Support:** System configuration and settings.

## 🚀 Tech Stack

*   **Frontend:** HTML5, CSS3 (Tailwind CSS), JavaScript
*   **Charts:** Chart.js / Recharts
*   **Icons:** Lucide Icons
